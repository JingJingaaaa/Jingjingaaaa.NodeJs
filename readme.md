# 如何使用
+ 克隆
```
$ git clone https://github.com/JingJingaaaa/Jingjingaaaa.NodeJs.git

```
+ 安装并运行
```
$ npm install
$ npm start

```
+ 建议使用WebStorm开发工具
+ 要确保您要有MongoDB数据库
+ 打开MyProject
    - 复制文件db路径
+ 打开命令窗口
```
$ mongod --dbpath="文件db的路径"

```
+ 打开MongoDB
    - 建立链接
+

+ 恭喜您可以使用啦（撒花）

+ "start": "node ./bin/www"
    - 在浏览器打开
    ```
    localhost:3000
    ```
